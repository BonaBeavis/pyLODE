Description

This ontology is an OWL interpretation of elements of the National Archives of Australia (NAA)'s Records Authorty documents and some of their corresponding NAA database elements. It is dependent on the CRS Ontology for overall context regarding Agency/Record associations and just models the additional RA elements the CRS ontology doesn't.

Disposal Actionc

IRI http://linked.data.gov.au/def/ra#DisposalAction
Description A Disposal Action is an instruction to carry out the disposal (retention or descruction) of a government Record. In addition to being a skos:Concept since it acts as a classifier, a Disposal Action is also a Provenance Ontology Plan meaning it instructs a course of action - how to dispose of a record.

prov:Plan c

Numbered Subclassc

IRI http://linked.data.gov.au/def/ra#NumberedSubclass
Description A numbered subclass is a finer-grained Record categorisation concept than an RA Class within a Records Authority. Numbeed Subclasses are idnetified in Records Authorities with an entry number and relate directly to Disposal Actions.
Super-classes skos:Concept c
In domain of http://linked.data.gov.au/def/ra#requires op

RA Classc

IRI http://linked.data.gov.au/def/ra#RaClass
Description An RA Class is a categorisation concept for collections of Records within a Records Authority. An RA class is equivalent to the CRS Ontology's Function records categorisation.
Super-classes skos:Concept c

Records Authortyc

IRI http://linked.data.gov.au/def/ra#RecordsAuthority
Description A Records Authority is a document that identifies minimum retention periods for Records and authorises the destruction of Records as required by Section 24 of the Archives Act 1983. A Records Authority individual, as represented in this ontology, contains a collection of RA Classes and Numbered Subclasses used to categorise Records in order for them to be assiged Disposal Actions.
Super-classes owl:NamedIndividual c
skos:ConceptScheme c
